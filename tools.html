Nathan Haley's Recommended Tools for Engineers

All 'Tool' credit goes to original site and content creators. Nothing in this directory of links represents anything I have ever worked on. My inspiration to include something like this on my website comes from the site Engineers-Excel.com

Software Engineering

General

Udemy: Excellent resource for learning many different software engineering topics via (often) discounted bootcamps/webinars

Head First Object-Oriented Analysis and Design: A very good book for learning object oriented software design methodology

Head First Design Patterns: A very good book for learning design patterns

C#

Formal comprehensive Microsoft C#/.NET documentation

C# Basics Cheat Sheet

Head First C#: A very good book for learning C#

Python

Formal comprehensive Python documentation

Python Basics Cheat Sheet

Head First Python: A very good book for learning Python

Unity

The official comprehensive Unity Manual

Unity Asset Store

Unity Framework for Turn Based Strategy Games

Artificial Intelligence

The latest from OpenAI

Deeplearning.ai: News and courses on AI topics

Sora: Latest OpenAI tool for AI generated videos

Web Design

Mozilla Developer Network (MDN): Web Design Resources

Coolors: Color Palettes, Styles, Hex Values, etc. handy for Web Design

Comprehensive HTML Unicode Character Sequence Code List

Cheat Sheet for the Emmet Visual Studio plug-in abbreviations for common HTML blocks

CSS Specificity Score Calculator: Useful for determining, which style properties prevail where conflicts exist

Mechanical Engineering

General

Up Codes: Excellent repository for State and Municipal construction codes with new AI CoPilot functionality

Engineering Toolbox: Offers condensed articles/reference tools for many different topics across various engineering topics

Engineers-Excel: Contains useful free Excel based apps useful for process control/PID tuning

PhET: Collection of many interactive software modules intended for demonstrating common physics concepts

Energy

Sustainable Energy Without the Hot Air: Great free online book on considerations regarding a sustainable future by means of energy balance accounting with sources and demands

Climate Change Indicators in the United States: Latest EPA report on how climate change impacts the world today and in the near future

Clean Building Performance Standard: Novel WA legislation intended on reducing the carbon footprint of the commercial built environment

Seattle's Building Emissions Performance Standard: Seattle legislation similar to Clean Building Performance Standard with added impetus for reduction of natural gas usage in buildings'

Seattle's Energy Benchmarking ordinance which requires commercial/multifamily building owners to benchmark energy usage annually using Energy Star Portfolio Manager

Energy Star Portfolio Manager: DOE created and recognized tool used for benchmarking of energy usage among other things in buildings

PSE's EnergyCAP: Puget Sound Energy's tool for integrating/automating customer energy usage data directly into Energy Star

SCL's Automated Benchmarking Service: Similar to EnergyCAP but for Seattle City Light. Same idea in concept, but alot clunkier in practice

HVAC

ECaps: Greenheck Fan and other dry-side HVAC apurtenance online selection software

ESP Systemwize: Pump and other hydronic (wet-side) HVAC appurtenance online selection software

Free read-only ASHRAE standards

Great article on smoke control systems for high/mid-rise HVAC building design

Carrier Block Load/HAP: Super useful HVAC software for thermal load calcs and energy modeling of HVAC systems

The Engineering Mindset: Great resource for understanding HVAC systems and concepts with good diagrammatic videos

TecTube: Great resource for understanding HVAC systems, concepts, and components with hands-on demos

Manufacturing

Product Design and Manufacture and Assembly: Very thorough and informative textbook on DFMA that covers a variety of manufacturing and assembly processes

DFM and DFA Software: Companion design software to the textbook above

The Toyota Way: Excellent book on the innovative and practical management principles used in the Toyota Production System that are highly applicable to manufacturing but also to other industries

Electrical Engineering

General

The National Electrical Code (NFPA 70) free online access

Mike Holt's Electrical Resources: Excellent repository of free online resources for understanding practical and theoretical electrictal power considerations

Informative and concise article on electrical equipment working clearances essential for layout/design

Free online crash course for HVACR engineers to understand relevant electrical concepts that tie into the world of HVACR equipment selection
